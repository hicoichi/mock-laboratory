# 理想人格RPG

## 仮説

「理想の自分」をゲームキャラクターとして具現化し、現実の行動でそのキャラクターを育てる体験を提供することで、疲れている日でも行動との接点を保ちやすくなる。

たまごっちのように「放置すると弱る」仕組みが適度なプレッシャーになり、継続のトリガーになるのではないか。

## 検証したいこと

- ユーザーは「理想の自分」というキャラクターに没入感を感じるか
- 現実の行動 → XP獲得 → 成長という変換が継続モチベーションになるか
- 接続が弱まる演出が、適切なプレッシャーとして機能するか（過剰でないか）

## 成功条件

インタビューで5人中3人以上が「これを使って行動を続けたい」と回答する。

## ユーザー体験フロー

1. 理想の自分に名前と説明をつける（例：「毎朝走る石川」「タフで賢い行動者」）
2. 目標カテゴリを選ぶ（学習・健康・創作・精神・仕事・発信）
3. 日々の行動をテキストで入力 → XP獲得 → レベルアップ
4. 連続記録でストリークが伸び、キャラクターが「絶好調」状態に
5. 記録を忘れると接続が弱まり、キャラクターが弱体化（たまごっち的）
6. ペルソナが一人称で語りかけてくる（没入感の核心）

## 技術

- HTML + CSS + Vanilla JS（単一ファイル）
- localStorage によるデータ永続化
- 外部依存なし

## 起動方法

```
index.html をブラウザで開く
```
